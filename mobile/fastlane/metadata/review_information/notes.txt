HumTrack works fully without sign-in. Tap the "작업 시작" / "Start" button on the home screen to record a vocal, analyze it, and edit the resulting multitrack (piano, bass, drums, vocal). All core features — recording, AI analysis, multitrack editing, pitch correction, quantize, MIDI/WAV preview — are available to anonymous users without an account.

Pro features (unlimited MIDI/WAV export, cloud sync) are gated behind in-app subscription. Please test purchases via Apple's Sandbox IAP testers; we have not created a custom demo account because OAuth-protected sign-in (Apple/Google) requires 2FA that would block the review flow.

Sign In with Apple / Google is optional and only used for the Pro cloud sync feature. Please test the app as an anonymous user unless explicitly verifying cloud sync — in which case Sandbox IAP after sign-in is the intended path.

Microphone permission is required for the recording feature; please grant it when prompted.
