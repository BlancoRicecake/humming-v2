HumTrack은 흥얼거린 멜로디 한 줄을 진짜 곡으로 만들어주는 가장 쉬운 모바일 DAW입니다. 허밍만 하면 MIDI·코드·멀티 트랙 세션이 자동으로 완성됩니다. 음악 이론도, 복잡한 학습도 필요 없어요.

처음 작곡을 시작하는 초보자, 길에서 떠오른 아이디어를 빠르게 메모하고 싶은 작곡가, 모바일에서 비트를 만들고 싶은 비트메이커까지 — HumTrack은 머릿속 멜로디를 바로 한 곡으로 만들어주는 작곡앱입니다.

【 무엇을 할 수 있나요 】
• 허밍 / 노래 / 악기 녹음 → 자동 음정 · 박자 · 코드 분석
• 멀티 트랙 편집: 보컬 + 피아노 + 베이스 + 드럼 + 신스
• 메트로놈과 박자 보정(Quantize)으로 정확한 타이밍
• 키 자동 감지로 코드가 항상 자연스럽게
• MIDI · WAV 내보내기 — 어떤 DAW로도 열 수 있고 바로 공유 가능
• 로컬 저장 — 로그인 없이 바로 시작

【 왜 HumTrack인가요 】
• 음악을 처음 만드는 사람을 위해 설계된 모바일 DAW이지만, 진지한 작곡에도 충분한 기능
• 허밍-MIDI 변환은 사라지기 전에 아이디어를 잡는 가장 빠른 방법
• 휴대용 음악 스튜디오, 비트메이커, 작곡 메모를 한 앱에서
