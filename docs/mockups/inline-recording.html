녹음 시작
  • 녹음 시작 pill 탭 직후 → 카운트다운 인라인으로 표시 (lane 중앙).
  • 3, 2, 1 큰 숫자가 차례로 강조됨. 활성 숫자 = 빨강 풀 크기 + 옅은 글로우, 지난 숫자 = 흐려짐.
  • 각 숫자 ~400ms (총 1.2초). 빠른 템포로 녹음 시작 흐름 끊지 않게.
  • 각 숫자 표시 시 가벼운 메트로놈/틱 사운드(짧은 high-tick) + 마지막 숫자 후 더 강한 다운비트 또는 햅틱.
  • 카운트다운 종료 즉시 Frame 3(녹음중) 으로 전환.
  • 카운트다운 중 lane border/배경은 이미 빨강(녹음 모드 진입). 상단 카드/다른 트랙은 dim.
  • 사용자가 카운트다운 중 lane 다시 탭하면 즉시 취소 → Idle 복귀.

녹음 시작
  • 녹음 중인 트랙 lane:
    • 좌측: 빨간 점 깜박임 + 경과 시간 (0:03)
    • 중앙: 실시간 amplitude 미터 (마이크 입력 진폭이 좌→우로 누적)
    • 우측: 빨강 정지 버튼(■). 탭 → 녹음 종료 + pending 시트.
  • 레인 시각: 빨강 border + 옅은 빨강 배경 + 미세 펄스 그림자. 녹음 중임을 강하게 표시.
  • 플레이헤드: 빨강(녹음 중)으로 색 변경. (일반 재생/편집은 흰색)
  • 상단 카드: 흐려짐 (disabled). 녹음 중엔 옵션 변경 안 됨을 시각화.
  • 다른 트랙: dim(opacity 0.4). 포커스가 녹음 트랙에 있음을 강조.
  • 재녹음도 동일 — 기존 청크 lane 안에서 같은 인라인 UI 가 나타남(청크가 잠시 숨겨지고, 종료 시 새 청크로 교체할지 시트로 묻기).
