트랙 루프 시각 표시

DRUM 트랙(원본 0~4s)이 looping=true → 가장 긴 다른 트랙(CHORDS, 0~14s)까지 자동 반복. 원본은 진한 청크, 반복본은 흐림/점선 처리.

옵션 A — 점선 반복 청크 (추천)
0:00
0:02
0:04
0:06
0:08
0:10
0:12
CHORDS
♪피아노 🔊
DRUM
🥁DRUM ↻ 루프 🔊
  • 원본 청크는 솔리드 lime tint + 솔리드 border.
  • 반복본은 opacity 0.55 + dashed border + 더 옅은 fill.
  • 트랙 이름 옆 ↻ 루프 뱃지로 명확히 표시.
  • 장점: 사용자가 "이게 반복되고 있구나" 한눈에 인지. 어디까지 반복되는지 정확히 보임.

  • 원본만 편집 가능 영역, 반복본은 read-only 한 느낌.
  • 단점: 노트가 가려져 조금 답답해 보일 수 있음.
