  • Lane = 전체 가로폭. 별도 sidebar 없음, 시각적 padding 없음.
  • 플레이헤드는 viewport-x = 86 위치에 흰색 세로선으로 고정. 룰러부터 마지막 트랙까지 한 줄로 관통.
  • 컨텐츠(청크/노트)는 scroll layer 안에 들어가며, scroll layer 의 left = (86 - scrollX). 따라서 scrollX=0 일 때 컨텐츠가 viewport-x=86 부터 시작.
  • 재생 시 컨텐츠가 좌측으로 흘러도 lane 안에 그대로 보이며, 좌측에 가려지는 영역은 없음(lane 좌측 끝까지 보임).
  • 녹음 시작 pill 은 lane 중앙.
