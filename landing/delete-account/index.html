계정 및 데이터 삭제

HumTrack 계정과 클라우드에 저장된 모든 작업물을 영구 삭제하는 방법입니다.

1. 앱 내에서 삭제하기 (권장)

  1. HumTrack 앱 실행 → 우상단 계정 아이콘 탭
  2. 설정 → 계정 진입
  3. "계정 삭제" 버튼 탭
  4. 확인 다이얼로그에서 "삭제 확인" 탭

삭제 요청은 즉시 백엔드로 전달되며, 72시간 이내에 모든 클라우드 데이터가 영구 삭제됩니다.

2. 이메일로 요청하기

앱에 접근할 수 없는 경우 다음 주소로 직접 요청해 주세요:

heobusy@gmail.com 로 요청

이메일에 다음 정보를 포함해 주세요:

  • 가입 이메일 주소 (Apple "Hide my email" relay 주소 포함)
  • 가입 방법 (Apple / Google)
  • 요청 사유 (선택)

처리 SLA: 영업일 기준 3일 이내 접수 확인, 72시간 이내 전면 삭제.

4. 법령상 보관되는 데이터

다음은 법령 의무에 따라 일정 기간 보관되며, 그 후 자동 폐기됩니다:

데이터보관 기간근거
IAP 결제 영수증 (transactionId, productId, 금액)5년「전자상거래법」, 「전자금융거래법」
IAP webhook 알림 로그90일분쟁 대응
서버 접근 로그 (IP, 요청 시각)90일보안·악용 방지

위 데이터는 개인 식별이 어려운 형태로 보관되며, 추가적인 마케팅·분석 용도로 사용되지 않습니다.

5. 무료 사용자의 데이터

로그인 없이 사용하는 익명 사용자의 경우 별도 삭제 요청이 필요하지 않습니다:

  • 보컬 녹음: 분석 서버 메모리에서 처리 후 분석 완료 즉시 폐기. 서버 디스크에 기록 ❌
  • 로컬 저장 프로젝트: 디바이스에만 저장. 앱 삭제 시 함께 제거됨 (iOS/Android OS 가 자동 처리)

6. 활성 구독 상태에서의 삭제

주의 — 계정 삭제는 진행 중인 유료 구독을 자동 해지하지 않습니다. 구독은 별도로 Apple App Store 또는 Google Play 의 구독 설정에서 직접 해지해 주세요:
  • iOS: 설정 → [사용자 이름] → 구독 → HumTrack
  • Android: Play 스토어 → 프로필 → 결제 및 정기결제

Account & Data Deletion

How to permanently delete your HumTrack account and all cloud-stored data.

1. Delete From the App (Recommended)

  1. Open HumTrack → tap the account icon (top-right)
  2. Go to Settings → Account
  3. Tap "Delete account"
  4. Confirm in the dialog

The request is sent to our backend immediately. All cloud data is permanently deleted within 72 hours.

2. Request by Email

If you cannot access the app, email us directly:

Email heobusy@gmail.com

Please include:

  • Your sign-up email (including Apple "Hide my email" relay address)
  • Sign-up method (Apple / Google)
  • Reason (optional)

SLA: Acknowledged within 3 business days, fully deleted within 72 hours.

3. Data That Is Deleted

DataStorageAction

Project metadata (title, BPM, tracks)Supabase PostgresDeleted immediately
Vocal files (Pro only)Cloudflare R2Deleted immediately
Usage counters (analyses, upload bytes)Supabase PostgresDeleted immediately

4. Data Retained for Legal Compliance

The following is retained as required by law, then automatically purged:

DataRetentionBasis
IAP receipts (transactionId, productId, amount)5 yearsKorean Electronic Commerce Act / Electronic Financial Transactions Act
IAP webhook logs90 daysDispute resolution
Server access logs (IP, request time)90 daysSecurity / abuse prevention

This data is stored in a form that minimizes identifiability and is not used for marketing or analytics.

5. Anonymous (Free) Users

If you have used HumTrack without signing in, no deletion request is needed:

  • Vocal recordings: processed in server memory and discarded immediately upon analysis completion. Never written to disk.
  • Local projects: stored only on your device. Removed when you uninstall the app (handled by iOS/Android).

6. Active Subscription Notice

Important — Deleting your account does not automatically cancel an active subscription. Cancel it separately in the App Store or Google Play subscription settings:
  • iOS: Settings → [Your Name] → Subscriptions → HumTrack
  • Android: Play Store → Profile → Payments & subscriptions
